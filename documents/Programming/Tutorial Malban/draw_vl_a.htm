Draw_VL_a ($F3DA)

This routine draws vectors between the set of (y,x) points pointed to by the register. The current scale factor is used. The vector list has the following format:

rel y, rel x, rel y, rel x, ...

 

ENTRY:

DP = $D0

A-reg = the number of vectors to draw

X-reg points to the vector list

 

EXIT:

X-reg points to next byte after list

D-reg trashed
